# DevOps-Workflow

- Set up environment on AWS; 
- Deploy application 'Hello World' in terms of container;
- Push image to ECR;
- Deploy and configure EC2 instance;
- Deploy NGINX solution (in terms of containers or not);
- Configure NGINX;
- Make the exposement of the application (optional: deploy a LB or not);
- Create a DNS record to the URL of the application (ezops.cloud);
- Do the tests of the solution
